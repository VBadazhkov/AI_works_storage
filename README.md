# Ноутбуки ML/AI

Этот репозиторий содержит различные проекты по искусственному интеллекту и машинному обучению по темам: глубокое обучение, LLM, RAG, prompt engineering, NLP-пайплайны.

## Структура хранилища

```
AI_works_storage/
├── CNN and MLP/              # Сверточные нейронные сети и многослойные перцептроны
├── fine tuning/              # Тонкая настройка моделей
├── NER/                      # Пайплайн распознавания именованных сущностей
├── prompt engineering/       # Техники промпт-инжиниринга
├── RAG/                      # RAG-пайплайн
└── text classification/      # Задачи классификации текста
```

## Обзор проектов

### 1. CNN и MLP (Сверточные нейронные сети и многослойные перцептроны)

Эта директория содержит проекты, сфокусированные на подходах глубокого обучения для задач регрессии:
- **Предсказание года**: Предсказывает год выпуска песен на основе аудио-фич с использованием архитектуры MLP
- **Предсказание цен на жилье**: Предсказывает цены на жилье с использованием многослойного персептрона

### 2. Тонкая настройка

Эта директория содержит эксперименты по тонкой настройке предобученных моделей.

### 3. Распознавание именованных сущностей (NER)

В ноутбуке реализуется идентификация и классификация именованных сущностей в текстах запросов пользователей маркетплейса. Ноутбук содержит:
- Реализацию NER-пайплайна
- Анализ n-грам
- Визуализацию результатов с примерами

### 4. Промпт-инжиниринг

Эта директория содержит реализации техник промпт-инжиниринга для больших языковых моделей:
- **Qwen3**: Используется модель Qwen3 с квантизацией
- **Техники промпт-инжиниринга**: Используются техники промптинга (например, CoT)
- **Оценка качества**: Модель прогоняется на бенчмарке

### 5. RAG

RAG-пайплайн для корпуса художественной литературы:
- **RAG-система**: Извлекает релевантную информацию из книг на основе запроса пользователя
- **Модель Qwen3-8B**: Используется квантизованная большая языковая модель

### 6. Классификация текста

Этот проект фокусируется на задачах классификации текстов социальной сети:
- **Анализ твитов**: Классификация твитов
- **Эмбеддинги твитов**: Используются для извлечения семантики в качестве дополнительного признака для Catboost
- **Метрики**: Для оценки качества классификации используется метрика PR AUC и confusion matrix
